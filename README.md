BOJ-Code-Downloader
=====================
[Baekjoon Online Judge](https://www.acmicpc.net/) 에서 제출한 소스코드를 한번에 다운로드합니다.

## 1. Run

```
$ python3 setup.py install
$ python3 download.py
```

* C/C++/Python/Java/Text로 작성 된 소스코드만 다운로드됩니다.
* 다운로드가 완료된 소스코드들은 "/download"에 저장됩니다.

## 4. Using
	1. Python3
	2. requests
	3. bs4
